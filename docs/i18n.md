# I18n

多语言能力默认关闭，默认语言为中文。配置入口在 `src/config/app.ts`，用户设置持久化在 `i18n-store`。

## 当前能力

- 默认支持 `zh-CN` 和 `en-US`。
- 页面文案通过 `useI18n().t(key)` 获取。
- 关闭多语言时，页面统一回退到 `zh-CN`。
- 已同步小程序原生导航标题和 tabBar 文案。
- “我的”页提供启用开关和语言选择。

## 配置项

```bash
TARO_APP_I18N_ENABLED=false
TARO_APP_DEFAULT_LOCALE=zh-CN
```

`TARO_APP_I18N_ENABLED=true` 只表示首次默认启用。用户在小程序内切换后，本地持久化设置会优先。

## 使用方式

```tsx
import { useI18n } from '@/hooks/useI18n'

export function Demo() {
  const { t } = useI18n()

  return <Text>{t('home.title')}</Text>
}
```

带变量的文案使用 `{name}` 占位：

```tsx
t('profile.currentUser', { nickname: 'Mock User' })
```

## 新增语言

1. 在 `src/i18n/index.ts` 的 `LOCALES` 中增加语言。
2. 为新增语言补齐 `I18nKey` 对应的全部文案。
3. 如果需要设置为默认语言，更新 `.env` 中的 `TARO_APP_DEFAULT_LOCALE`。

新增 key 时需要先加到中文字典，英文和其他语言会被 TypeScript 要求补齐，避免漏翻。
